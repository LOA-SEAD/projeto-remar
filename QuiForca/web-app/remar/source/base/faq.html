QuiForca - Acessível

  • Estilo: Jogo de forca
  • Objetivo Pedagógico: Cobrar conhecimentos relacionados à química, buscando palavras e ajudas de um banco de dados externo e realimentável.
  • Tecnologias Utilizadas: HTML5, Javascript, CSS e diretrizes de acessibilidade WAI-ARIA.
  • Resumo: Como em um jogo de forca, é cobrada uma palavra como resposta dada uma certa dica relacionada.
